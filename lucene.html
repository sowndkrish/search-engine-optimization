1. Sort By Title

The results are sorted for faster identification of the search result. Since, the hyperlinks contain titles of academic papers, it is essential to sort by alphabetic order inorde to locate the name of the file.

2. Content display around title

The search result will insert html tags such as p or span around the content which appears as part of the search and displays in the web page. The color and location can be controlled.

3. Pagination

When we are facing the necessity to display large amount to data, pagination is one of the option. To enable faster pagination, we use JQuery to split our content into several pages. For small or middle datasets, we can include pagination at the client side instead of server side to reduce the load. It can be implemented using JQuery and JavaScript.

Sorting the Search Results

Based on the input from the webpage, the search is performed and returned the titles. This needs to be sorted to produce better results for display and locating the search result more faster. At the client end, javascript performs sorting by titles and displays in the webpage.

Validating at the server end
			String title = doc.get("title");
			Pattern pt = Pattern.compile("[^a-zA-Z0-9\\s]");
			Matcher match = pt.matcher(title);
			while (match.find()) {
				String s = match.group();
				title = title.replaceAll("\\" + s, "");
			}
			if (title != null) {}
  • Checks if the title is not null and contains only alphanumeric characters at the server end, to eliminate any special characters.

Relevant Text around Hyperlinks

The search result yields only the titles of the conference papers in the search result. It is essential to display some information around the search to make the users look for their relevant searches. The hyperlink leads to the search file which the content around it gives the users more accurate results.
